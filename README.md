# Fraud Detection Using Machine Learning

Master's Capstone Project focused on identifying potentially fraudulent financial transactions using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## Features
- Fraud risk monitoring dashboard
- Behavioral pattern analysis
- Concept drift monitoring
- Explainable AI insights
- Exportable fraud reports
